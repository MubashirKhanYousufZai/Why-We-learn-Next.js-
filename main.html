Why we learn Next.js ?

Next.js simplifies the process of building a react application fot production.

Features Of Next.js

  1. File Based Routing
  2. Pre-Rendering
  3. API-Routes
  4. Support For CSS modules
  5. Authenticaion(auth)
  6. Dev and Prod build system

1. File Based Routing:

In React we need to install third party routing packages, configuration and add code.
But Next.js provide file based routing which make this process easier for us.

2. Pre-Rendering:

Next.js genrates HTML for each page in advance Instead of Client Side Java Script.
& This improves the SEO that helps us our website to hit the GOOGLE's SEO.
3. API-Routes:
We can create API routes by using next.js. So next.js is a ful stack framework
If you think about it we can write the frontend code in react but also write api's that can called by the react app.
4. Support for CSS modules:
Next.js supports CSS modules which saves the time of choosing css library.
Offcourse we are allowed to use the css framework like Tailwind or even a CSS
in js library like style components for example but Next.js supports Css modules out of the box.
 5. Authentication(AUTH):
 Next.js supports multiple authenticaion pattrens each designed for different use cases.
 6. Dev and Prod build system:
 Next.js gives dev and build system and a well optmised production system.
so that we can concentrate on our code more than the configuration.
It is a culimination of these features which make next.js an amazing framework to work.
